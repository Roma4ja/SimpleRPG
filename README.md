# SimpleRPG ⚔️

Консольная RPG игра на языке C#. Создай персонажа, сражайся с врагами и выживи!

## Скачать
Скачай последнюю версию в разделе [Releases](https://github.com/Roma4ja/SimpleRPG/releases) — установка не нужна, просто запусти `.exe`

## Геймплей
- Введи имя своего персонажа
- Выбирай действия в бою — атаковать, вылечиться или выйти
- Сражайся с врагами в пошаговых боях
- Побеждай противников и выживай

## Управление
1 — Вылечиться  
2 — Сразиться  
3 — Выйти  

## Структура проекта
- `Program.cs` — точка входа, главное меню
- `Player.cs` — класс игрока (HP, атака, защита)
- `Enemy.cs` — класс врага
- `Battle.cs` — логика пошагового боя
- `Character.cs` — базовый класс персонажа

## Технологии
- Язык: C#
- Платформа: .NET 10
- Тип: консольное приложение
- IDE: Visual Studio

## Системные требования
- Windows 10/11 x64
- Установка не требуется
